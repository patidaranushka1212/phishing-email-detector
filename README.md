# 🔐 Email Spam & Phishing Detector

A machine learning-based web application that detects whether an email is **safe or phishing**.

## 🚀 Features
- Detect phishing emails using trained ML model
- Supports full email content input
- Clean UI with blue-white professional design
- Built using Flask

## 🧠 Tech Stack
- Python
- Flask
- Scikit-learn
- HTML/CSS
